BiblioTech

BiblioTech é um sistema onde você pode monitorar o uso indevido dos computadores dentro das bibliotecas.

Controle de segurança

O monitoramento dos hardwares nas bibliotecas é uma função muito importante para o controle de atividades. Essencialmente utilizada para estudos, as bibliotecas têm algumas regras a serem seguidas durante o uso dos computadores. Instalação de jogos e/ou programas que podem comprometer o bom funcionamento da máquina são proibidos no local, mas como faríamos para controlar a utilização inadequada do computador?
